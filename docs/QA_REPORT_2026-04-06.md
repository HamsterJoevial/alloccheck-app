## P0 -- Bloquants

### QA-001 -- Tests echouent sur mobile (annotations @JS)
- **Type** : Bug
- **Fichier** : `lib/core/services/payment_service.dart:9-14`, `lib/features/results/screens/results_screen.dart:15`, `lib/features/letter/screens/letter_screen.dart:14`
- **Ce qui est vu** : Les annotations `@JS()` et `external` sont utilisees sans condition de compilation. Sur mobile, le compilateur echoue.
- **Impact** : Impossible de compiler les tests, potentiellement impossible de builder sur mobile.
- **Suggestion** : Isoler le code JS dans des fichiers conditionnels via exports conditionnels.

### QA-002 -- Paywall contournable trivialement
- **Type** : Faille securite
- **Fichier** : `lib/core/services/payment_service.dart:24,31,62`
- **Ce qui est vu** : Token `AC2026UNLOCK` en dur dans le code. `unlockWithCode()` compare simplement le code saisi. Etat de deblocage en clair dans SharedPreferences/localStorage.
- **Impact** : N'importe qui peut debloquer le rapport sans payer.
- **Suggestion** : Validation serveur du paiement Stripe.
